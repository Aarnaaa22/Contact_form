# Aarna's Contact Form

A simple contact management web application built with HTML, CSS, and JavaScript.

## Project Description

This is a dynamic contact form that allows users to add, edit, delete, and search through contact entries. All data is stored in the browser's localStorage, so contacts persist even after the page is refreshed.

## Features

- **Add Contacts**: Enter name, email, and phone number to add new contacts
- **Edit Contacts**: Click the Edit button on any contact to modify their information
- **Delete Contacts**: Remove contacts with the Delete button
- **Search Functionality**: Filter contacts by name or email in real-time
- **Drag and Drop**: Reorder contacts by dragging and dropping them
- **Data Persistence**: All contacts are saved in localStorage and remain after page refresh
- **Form Validation**: Ensures all fields are filled before adding a contact
- **Clean UI**: Simple lavender/purple themed design

## Files Included

1. **index.html** - Contains the HTML structure of the contact form
2. **style.css** - Contains all styling with a lavender/purple color theme
3. **script.js** - Contains all JavaScript functionality including:
   - Adding, editing, and deleting contacts
   - Search/filter functionality
   - Drag and drop reordering
   - localStorage for data persistence

## How to Use

1. **Download all 3 files** (`index.html`, `style.css`, `script.js`)
2. **Place them in the same folder** on your computer
3. **Open `index.html`** in a web browser (Chrome or Edge recommended)
4. Start adding your contacts!

### Adding a Contact
- Fill in the Name, Email, and Phone Number fields
- Click the "Add Contact" button
- The contact will appear in the list below

### Editing a Contact
- Click the "Edit" button on any contact card
- The form will populate with that contact's information
- Make your changes and click "Save Changes"

### Deleting a Contact
- Click the "Delete" button on any contact card to remove it

### Searching Contacts
- Type in the search bar at the top
- Contacts will filter automatically as you type
- Search works on both name and email

### Reordering Contacts
- Click and drag any contact card to reorder the list

## Technical Details

- **HTML5** for structure
- **CSS3** for styling (no external frameworks)
- **Vanilla JavaScript** for functionality (no libraries)
- **localStorage API** for data persistence
- **Drag and Drop API** for reordering
